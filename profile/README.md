# Elmedia Player — Universal Player With Support for Any Format and AirPlay

## About Elmedia Player

Elmedia Player provides a universal media player for macOS that plays virtually any video or audio format without additional codecs — AVI, MKV, MP4, MOV, FLV, WMV, MP3, FLAC, AAC, OGG, and dozens more. Elmedia Player combines hardware-accelerated decoding (low CPU usage), built-in AirPlay receiver and sender (stream to Apple TV or receive from iPhone/iPad), online subtitle search (OpenSubtitles integration), playback speed control (0.5x to 4x), and audio equalizer. Elmedia Player enables users to watch movies, listen to music, and stream content to or from Apple devices. Elmedia Player is trusted by users who need reliable playback of obscure formats and seamless Apple ecosystem integration.

## Deep Dive — Universal Media Player

Format support — video (AVI, MKV, MP4, MOV, FLV, WMV, M4V, 3GP, ASF, DIVX, MPEG, VOB), audio (MP3, FLAC, AAC, OGG, WAV, M4A, AC3, DTS, AIFF, APE), subtitles (ASS, SRT, SSA, SUB, IDX)  
AirPlay features — AirPlay sender (stream from Elmedia to Apple TV, smart TV, AirPlay speakers), AirPlay receiver (receive streams from iPhone, iPad, other Macs)  
Performance — hardware decoding (VideoToolbox), smooth 4K playback, minimal CPU usage, upscaling  
Subtitle features — online search (OpenSubtitles), manual sync adjustment (delay/advance by milliseconds), customize font, size, color, background  
Audio features — graphic equalizer (10 bands), audio effects (reverb, bass boost, vocal booster), sound normalization, volume boost (up to 200%)  
Playback controls — speed control (0.5x–4x), A-B loop repeat, bookmark scenes, frame-by-frame advance, screenshot capture  
PRO features ($19.99 lifetime) — AirPlay receiver, online subtitles download, YouTube playback (without browser), advanced audio options, customer support

## Quick Reference for Elmedia Player

| Operation | Elmedia Player Action |
|-----------|----------------------|
| Open file | Cmd+O |
| Full screen | Cmd+F |
| AirPlay (sender) | Click AirPlay icon → Select device |
| Enable AirPlay receiver | Preferences → AirPlay → Enable Receiver |
| Search subtitles online | Subtitles menu → Search → Enter movie name |
| Adjust subtitle sync | Subtitle menu → Sync +0.5s / -0.5s |
| Change playback speed | Playback menu → Speed → 0.5x to 4x |
| A-B loop repeat | Click A and B buttons on timeline |
| Take screenshot | Cmd+S |
| Open equalizer | Window → Equalizer |
| Open preferences | Cmd+, |
| Quit Elmedia Player | Cmd+Q |
